# AgriTrace-Blockchain-Based-Supply-Chain-System
Developed a blockchain-enabled agricultural supply chain platform for end-to-end product traceability.
Implemented role-based access control for farmers, distributors, retailers, and consumers.
Designed QR-code verification for real-time product authentication.
Built admin dashboard and analytics portal for supply chain monitoring.
Utilized hybrid on-chain and off-chain storage architecture for scalability.
